# sd
Search for and run shell scripts

```
USAGE:
    sd [OPTIONS] <FILE> [ARGS]...

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -d, --dir <DIRECTORY>    set root directory for this search

ARGS:
    <FILE>       file to search for
    <ARGS>...    arguments to pass the script
```
